模型设计

用点和线勾勒出案件之间的千丝万缕

  • 图: 无向图
  • 节点: 案件 ( 若案件已有逮捕犯人则放弃该节点 )
  • 边: 案件之间的距离 * 时间差 * 案件相似度
  • 若两件案件距离与案发时间不符 ( 二案发地点行程速度大于200km/h), 则去除该边
  • 案件之间的距离: 大圆距离
  • 通过经纬度和半正矢公式计算, 以千米作为单位
  • 时间差: 两件案件发生时间的差 ( 以小时作为单位 )
  • 案件相似度: 若案件的主要描述一样，案件相似度为1；否则为10

最终 权重大于20 的边不予考虑。

以上的限制条件所减去的边数量太少（<1%），总边数接近 n^2。
